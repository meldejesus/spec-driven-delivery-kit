# Messages

This folder is the optional durable output home for the message workflow.

Use it when a rewritten message is important enough to preserve beyond the chat session.

Recommended shape:

```text
workflow/messages/<message-name>/
  approach.md
  outline.md
  draft.md
  review.md
  final.md
  lessons-learned.md
```

For quick messages, keep the workflow conversation-only and do not create files.

Reusable style lessons can be proposed for `workflow/messages/style-lessons.md`.
